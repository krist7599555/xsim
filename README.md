# xsim

xcode device simulate cli

```text
Xcode simulator helper

Usage:  xsim [command]
      * xsim open
      * xsim kill
      * xsim restart
        xsim boot [search?]
        xsim ls
        xsim screenshot
        xsim booted

Options:
    --help   show documentation

See 'xsim --help' for more information on a specific command.

```
